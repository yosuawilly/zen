Niat Shalat


Niat Shalat Shubuh

Ushallii fardhu shubhi raka'aatini mustaqbilal qiblati adaan-an [makmuuman / imaaman] lillaahi ta'aalaa

{ Saya niat shalat Subuh dua rakaat menghadap kiblat pada waktunya (menjadi makmum / imam) karena Allah semata.}


Niat Shalat Maghrib

Ushalli fardhal maghribi tsalaatsa raka'aatim mustaqbilal qiblati adaa-an (ma'muman/imaman) lillaahi ta'aalaa

{ "Aku sengaja sholat fardlu maghrib tiga raka'at menghadap qiblat (sebagai imam/sebagai makmum) karena Allah". }


Niat Shalat Isya

Ushalli fardhal isyaa-i arba'a raka'aatim mustaqbilal qiblati adaa-an (ma'muman/imaman) lillahi ta'aalaa

{ "Aku sengaja sholat fardlu isya empat raka'at menghadap qiblat (sebagai imam/sebagai makmum) karena Allah" }


Niat Shalat Dzuhur

{ "Aku sengaja sholat fardlu dhuhur empat raka'at menghadap qiblat (sebagai imam/sebagai makmum) karena Allah". }

{ "Aku sengaja sholat fardlu ashar empat raka'at menghadap qiblat (sebagai imam/sebagai makmum) karena Allah". }
